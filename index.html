Tic-Tac-Toe

Rules of the Game

  1. The game is played on a grid that's 3x3 squares.

  2. The game has 2 Players: Player X and Player O.

  3. The game begins with Player X then followed by Player O. Players take turns putting their marks in the empty squares.

  4. The first player to get 3 of their marks in a row (horizontally, vertically, or diagonally) is the winner.

  5. When all of the squares are full, the game is automatically over. If no player has 3 marks in a row, the game ends in a draw.
